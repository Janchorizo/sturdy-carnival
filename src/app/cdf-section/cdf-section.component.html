Assigning ranks

using an Empirical Cumulative Distribution Function

Grade boundaries are better assigned using the ECDF to find the cuts, as it allows to easily find distinguishable groups of marks.
We can then search for cuts that would make a student's grade the furthest from the next rank.

Setting the score cuts for the marks. There are four different eligible marks (A, B, C, D, E) that are assigned to an exam result based on the boundaries you set.
